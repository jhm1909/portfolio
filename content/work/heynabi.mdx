## The Challenge

International students miss critical lecture content because real-time translation tools are either too slow, inaccurate, or not designed for academic contexts. Existing solutions fail to handle technical vocabulary and produce translations with unacceptable delays.

## The Solution

Created a specialized platform using Soniox for high-accuracy speech-to-text and Gemini 2.5 Flash for context-aware academic translation with under 2-second latency. The system is purpose-built for lecture environments, handling technical terms and academic language with precision.

## The Result

Achieved 99% STT accuracy with sub-2s translation latency, supporting 5 languages for live lectures, study sessions, and group projects. Students can now follow along in real-time without missing critical information.

## Key Features

- **Real-time speech-to-text** with 99% accuracy via Soniox
- **AI translation** powered by Gemini 2.5 Flash
- **5 languages:** Korean, Chinese, Japanese, Vietnamese, English
- **Under 2-second** translation latency
- **Google Sign-in** authentication
